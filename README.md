# React Web Utilities

A minimal, offline-first collection of web utilities and helper tools for daily developer needs. All conversions, parsing, and formatting run entirely client-side in the browser.

## Tech Stack

- React
- TypeScript
- Vite
- TailwindCSS
- Shadcn UI / Radix UI

## Getting Started

### Prerequisites

Ensure you have Node.js installed on your system.

### Installation

Install the project dependencies:

```bash
npm install
```

### Running Locally

Start the local development server:

```bash
npm run dev
```

### Building for Production

Compile and build the optimized production assets:

```bash
npm run build
```
